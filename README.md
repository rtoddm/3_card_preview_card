# 3-Column Preview Card Component

## Overview

This is a small front-end practice project built using Bootstrap.  
The goal of the project was to practice responsive layout, grid usage, and Bootstrap utility classes in a real UI scenario.

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5

## What This Project Demonstrates

- Use of the Bootstrap grid system for responsive layouts
- Mobile-first design approach
- Utility classes for spacing, alignment, and typography
- Clean, semantic HTML structure

## Notes

This project was created as a learning exercise while studying Bootstrap.  
It is intentionally simple and focused on layout and responsiveness rather than complex JavaScript or backend functionality.
